    DISEÑO Y ESTRUCTURA

    web, a tu navegador le llega este código, y lo traduce de forma visual para que veas lo que el creador de la web ha diseñado que puedas ver. Este código es el que decide la estructura una página web, colocando sus diferentes elementos en los punto correspondientes, y un fallo dentro de él hará que no se vea bien.
     

    QUE CAMBIA EN HTML5

    Este nuevo estándar no tiene una cantidad asombrosa de novedades, pero las que tiene son absolutamente revolucionarias. La más destacada es la posibilidad de añadir archivos multimedia a la web, como vídeos o audios, y que estos no tengas que insertarlos utilizando otros plug-ins. Todo está incluido dentro del código.
